# Keyboard Shortcuts

This page documents all keyboard shortcuts available in Eclipse Peon AI.

## Chat Input

| Shortcut | Action |
|----------|--------|
| **Ctrl+Enter** / **Cmd+Enter** | Send message (adds to history if AI is working) |
| **Ctrl+Z** / **Cmd+Z** | Undo text edit |
| **Ctrl+Shift+Z** / **Cmd+Shift+Z** | Redo text edit |
| **Ctrl+A** / **Cmd+A** | Select all text in input field |

## Slash Commands (when popup is open)

When typing a `/` command, the following shortcuts are available:

| Shortcut | Action |
|----------|--------|
| **↑ Arrow Up** | Navigate up through commands |
| **↓ Arrow Down** | Navigate down through commands |
| **Enter** | Commit selected command (without sending message) |
| **Tab** | Commit selected command |
| **Escape** | Hide command popup |

## Context Menu (Right-click on input)

The chat input supports a full context menu with:
- Undo
- Redo  
- Cut
- Copy
- Paste
- Select All

## Chat View

| Shortcut | Action |
|----------|--------|
| **Ctrl+=** / **Cmd+=** | Increase font size |
| **Ctrl+-** / **Cmd+-** | Decrease font size |
