# Chat Interface

The Peon AI chat view provides an interactive interface for communicating with your LLM.

## Opening the Chat View

1. Go to **Window > Show View > Other...**
2. Search for "Peon AI" or "AI Chat"
3. Click OK

## Sending Messages

- Type your message in the input field at the bottom
- Press **Enter** to send (Shift+Enter for new line)
- The AI response will appear in the chat

## Features

- **Syntax Highlighting**: Code blocks are highlighted based on language
- **Markdown Rendering**: Supports headings, lists, links, and more
- **File References**: Click on file paths to open them
- **Error Display**: Errors are shown in red with details

## Example Commands

- "What does the selected file do?"
- "Write a unit test for class X"
- "Explain this error: [paste error]"
- "Refactor this code: [paste code]"
